## 使用方式:

### 圖形介面:

``` Rectangle ``` 可畫出正方形
``` Circle ``` 可畫出圓形
``` Triangle ``` 可畫出三角形
``` Fill Color ``` 可畫出該圖形深色模式

### 功能:

``` Brush ```  畫筆
``` Eraser ``` 橡皮擦

``` Clear Canvas ``` 全部清除
``` Save As Image ``` 儲存並下載圖片
